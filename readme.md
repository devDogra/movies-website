
HOW TO RUN:
1) After cloning the repo, activate the virtual environment `. .venv/bin/activate`
2) Install dependencies (requirements.txt, package.json)
3) Run `python manage.py runserver`
4) Cd into frontend/
5) 'ng serve -o
